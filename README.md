# Анализ датасета Online Retail

## Данные
- **Источник:** Kaggle
- **Строк:** 541 909
- **Столбцы:** InvoiceNo, StockCode, Description, Quantity, InvoiceDate, UnitPrice, CustomerID, Country

## Что сделано
- **Очистка:** удалены строки без CustomerID, создан столбец TotalPrice = Quantity × UnitPrice
- **Анализ:** группировки по странам, месяцам, дням недели, товарам
- **Корреляции:** связь Quantity, UnitPrice и TotalPrice
- **Визуализация:** 6 интерактивных графиков (Plotly)

## Графики
1. Топ-10 стран по выручке
2. Динамика продаж по месяцам
3. Распределение размера корзины
4. Корреляционная матрица
5. Топ-10 товаров по выручке
6. Выручка по дням недели

## Выводы
- Основной рынок — Великобритания
- Пик продаж — ноябрь
- Самый популярный товар: WHITE HANGING HEART T-LIGHT HOLDER
- Выручка сильно зависит от количества, слабо — от цены

## Использованные библиотеки
pandas, matplotlib, plotly, seaborn
